# highlighter_userscript
simple chrome userscript to add text highlighting functionality.

- `alt + h` *or* `ctrl + shift + h`  to turn on/off highlighter
- press keys to change color:
  ```
            'r': Soft Coral
            'y': highlighter Yellow
            'g': Mint Green
            'b': Sky Blue
            'p': Light Pink
            't': Turquoise
            'l': Lavender
            'm': Pale Green
            's': Light Steel Blue
  ```
